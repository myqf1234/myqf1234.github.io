安装NOI Linux 2.0

如果您已经安装完成或只想了解NOI Linux 2.0的体验，请跳过此段。

安装镜像下载链接：https://noiresources.ccf.org.cn/ubuntu-noi-v2.0.iso

我在家里面下载只用了4分钟(网速太快了，12.5MB/S)，估计一般下载需要30分钟~1小时左右，网速慢一点的可能超过1小时，所以建议大家提前准备下载好，以免浪费太多的时间。

检查电脑规格：

  1. 你的电脑必须是64位操作系统，且必须是基于x64的处理器。
  2. 最重要的，网上许多教程没讲到的地方，你电脑的CPU必须支持并开启虚拟化，如果你的电脑为Mac，则无需检验此项。教程网上有很多，请自行搜索，本文不再多赘述。
  3. 你的电脑的运行内存(机带RAM)建议至少8GB，16GB则更好，可用磁盘空间至少30GB以上。

如何检查电脑的配置规格呢？我将以Windows10操作系统演示，其他系统操作方法差不多。

Windows：按下Windows+R键打开运行，输入ms-settings:打开系统设置，依次点击系统→关于，找到设备规格，查看即可。然后双击桌面上的此电脑，检查剩余磁盘空间是否大于30GB。

请尽量按照建议配置规格操作，否则可能会安装失败或发生更严重的问题。

安装虚拟机：

本人推荐VMware Workstation 16 Pro，其特点为使用简单方便，功能齐全，可以在https://www.vmware.com/cn/products/workstation-pro.html下载。

正式安装！Yes！

打开VM16 Pro，选择创建新的虚拟机，在新建虚拟机向导中选择典型，点击下一步。

之后选择稍后安装操作系统，不要选错了，否则会开启简易安装，从而导致安装不成功，我之前就犯过这个错误，怎么也安装不成功，点击下一步。

之后选择要安装的操作系统，系统选Linux，版本选Ubuntu(NOI Linux 2.0是基于Ubuntu 20.04.1定制的)，点击下一步。

之后为虚拟机命名，名字自己随便起，我这里就起NOI Linux 2.0吧。路径尽量选择系统盘以外的盘符，例如D盘你的系统盘在D盘当我没说，虚拟机是很占空间的，系统盘太满不是好习惯，点击下一步。

选择磁盘容量可以比建议容量大一些，比如我就选择了40GB，存储方式个人比较喜欢单个文件存储。如果你的磁盘空间比较紧张，可以选择20GB容量，如果经常在多台电脑之间移动文件，个人推荐拆分存储，点击下一步。

再次检查配置，没有问题就可以点完成了！

安装系统需要配置镜像才能启动。在虚拟机页面，点击编辑虚拟机设置，依次点击CD/DVD，使用ISO映像文件，浏览，找到刚刚下载的镜像，点击确定，保存设置。

终于到了激动人心的开启虚拟机时刻了！等待虚拟机加载片刻，便会跳出安装页面，将语言调整为简体中文，就可以安装了，点击安装Ubuntu。

键盘布局不用改了，就使用默认的Chinese-Chinese就可以了，点击继续。

更新和其他软件时，安装就选正常安装，注意，一定要勾选掉安装Ubuntu时下载更新，否则需要安装很长时间，点击继续。

不知道为什么VMware Tools自己安装上了安装类型就选清除整个磁盘安装Ubuntu，毕竟是虚拟机嘛，无所谓了，如果你要安装双系统用，请不要选择此选项，点击现在安装。

提示将以下改动写入磁盘吗，点击继续就可以了。

提示您在什么地方时，点击中国所在的位置，提示shanghai就成功了，点击继续。

提示您是谁时，输入自己的用户名和密码，不用多讲，自己定，我这里就选noilinux吧。

接下来就是安装时间了，等待一会，休息一下吧~

6个小时以后，终于好了，这时间都能一边慢慢悠悠聊天嗑好几包瓜子了！点击现在重启。

之后出现一个提示，按下ENTER键就可以了。

重启之后，输入用户密码。

大功告成！进入桌面，自此安装部分结束。

NOI Linux 2.0的体验

如果你只想看NOI Linux 2.0的安装部分，你的阅读可能到此结束了。

但是桌面只是一张奇丑无比的黑壁纸，右下角一个孤零零的NOI图标，对于我们来说怎能满意呢。可以点击鼠标右键，更换壁纸来更换。
