reverse(s.begin(),s.end());：reverse：定义在<algorithm>中，指将字符串、数组等反转，s.begin和s.end是两个广义指针，分别指向s的开头和末尾。

介绍一下getchar:

getchar原本的意思指读入一个字符，在这里可以把两个字符串之间的空格/回车”吃掉“，示例如下：

之后怎么办？输出就好了。

所以：

CODE：：

#include<iostream>
#include<string> //string类型用
#include<cstdio> //geichar()用
#include<algorithm> //reverse()用
//并不向大家推荐使用万能头，我们要明白每个头文件有什么用途。
using namespace std;
int main(){
    string s;
    while(cin>>s){
        reverse(s.begin(),s.end());
        cout<<s;
        char ch=getchar();
        cout<<ch;
    }
    return 0;
}
